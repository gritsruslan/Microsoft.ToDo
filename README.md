# Microsoft.ToDo

This is a small web application that replicates basic functionality of Microsoft To-Do.

The application allows users to:

- Create categories (task groups)
- Create, edit, and delete tasks
- Assign tasks to categories
- Search tasks by title or filters

Also users can register and sign in to access their own tasks and categories.

Tech Stack:

- Frontend: Angular 19 + Bootstrap
- Backend: .NET 10 + ASP.NET Core Web API
- Database: Microsoft SQL Server 2022

This is an educational project and is not intended for production use.
