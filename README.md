# Microsoft.ToDo

This is a small web application that replicates the basic functionality of Microsoft To Do.
The main goal of this project is to gain experience in the full software development lifecycle - from idea and implementation to deployment on Azure with CI/CD.

The application allows users to:

- Create categories (task groups)
- Create, edit, and delete tasks
- Assign tasks to categories
- Search tasks by title and filters

Users can also register and sign in to manage their own tasks and categories.

Tech Stack

- Frontend: Angular 19 + Bootstrap
- Backend: .NET 10 + ASP.NET Core Web API
- Database: Microsoft SQL Server 2022 (local), Azure SQL Database (cloud deployment)
- Cloud / DevOps: Azure App Service, Azure Static Web Apps, GitHub Actions CI/CD

This is an educational project and is not intended for production use.
